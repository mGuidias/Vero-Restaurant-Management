# Vero - Sistema de Gestão e Checkout

Vero - Sistema de Gestão para Restaurantes 🍽️
O Vero é um software de PDV (Ponto de Venda) e gerenciamento de comandas desenvolvido para otimizar o atendimento e o fechamento de contas em restaurantes, bares e lanchonetes. O foco do projeto é oferecer agilidade operacional com uma interface moderna e intuitiva.

🚀 Funcionalidades de Destaque
Check-out Inteligente: Sistema de pagamento com suporte a fracionamento de itens e divisão de conta entre múltiplos clientes.

Agilidade Operacional: Interface focada em produtividade com atalhos de teclado e seleção rápida de itens.

Interface Moderna (UI/UX): Design em Dark Mode com visual arredondado, focado na experiência do usuário.

Instalação Simples: Sistema leve que utiliza persistência de dados local, funcionando sem a necessidade de instalar bancos de dados complexos.

🛠️ Tecnologias
Linguagem: C# (.NET)

Framework: Windows Forms

Arquitetura: Organizada para garantir fácil manutenção e expansão do código.

📥 Como Testar (Sem precisar programar)
Se você não é desenvolvedor e quer apenas testar o sistema:

Vá até a seção Releases (no lado direito desta página).

Baixe o arquivo .zip da versão mais recente.

Extraia os arquivos e execute o Vero.exe.

Nota: O projeto inclui um arquivo cardapio_teste.xlsx que serve de base para os produtos do sistema.
